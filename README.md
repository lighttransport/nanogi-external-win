
nanogi-external-win
====================

A repository for prebuilt external libraries of ``nanogi``.  
All dependencies are built for VS2013 x64.

Dependencies
--------------------

- glm 0.9.5.4
- assimp rev. 3086569
- FreeImage-3.15.4
- Embree2 rev. b884189
- yaml-cpp 0.5.1
- ctemplate rev. 6742f62
- Eigen 3.2.5
